# Notification App Backend

This folder is part of the required repository structure for the Campus Hiring Evaluation.

As per the user's instructions, this submission follows the **Frontend Track**. Therefore, the primary implementation is located in the `notification_app_fe` and `logging_middleware` folders.

The backend for this application is expected to:
1. Register users with the Test Server.
2. Authenticate and obtain tokens.
3. Manage and dispatch notifications to various channels.
4. Integrate with the centralized logging service.
